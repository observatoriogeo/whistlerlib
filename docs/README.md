## Editing

- **Concept pages, install pages, intro, API stub, migration**: edit the file directly.
- **Tutorial pages** (`tutorials/01-*.md` etc.): **do not edit directly.** They're auto-generated from `examples/<slug>/README.md` by [`scripts/sync_tutorials.py`](../scripts/sync_tutorials.py). Edit the source example README, then run:
  ```bash
  python scripts/sync_tutorials.py
  ```
  CI runs `--check` and fails on drift.
